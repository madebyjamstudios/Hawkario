# Hawkario

Professional countdown timer with customizable display for broadcasts and presentations.

## Getting Started
1. Run `npm start` to launch the app
2. Click **Open Output Window** to open the display window
3. Press **F** in the output window to go fullscreen

## Features
- Countdown and count-up timer modes
- Customizable typography (font, size, weight, color)
- Warning system with color change and flash effects
- Preset management with import/export
- Real-time preview
